# Gastrodon Pack Revival 030

## 版本 Version: V0.5.0

## 特色 Features:
- 怪物: Vanilla + 6 | Enemies: Vanilla + 6
- 地圖: Vanilla + 5 | Maps: Vanilla + 5
- 模組數量 mod number: 94
- 可以完整遊玩，歡迎自行啟用移除模組，推薦使用 "RepoAdminMenu by proferabg" 模組進行設定作弊。(注意使用該模組時，可能會使 Enemy And Valuable Spawn Manager by soundedsquash 地圖顯示錯誤)

The game is fully playable. You are welcome to enable or disable mods as needed. We recommend using the "RepoAdminMenu by proferabg" mod for cheat settings. (Note: Using this mod may cause map display errors with Enemy And Valuable Spawn Manager by soundedsquash)

<details>
<summary><h2>已知 Bug | Known Bugs</h2></summary>

  - [PrivateLateJoin] 剛加入的玩家可能會有顯示問題或無法語音說話，直接進入下一關通常可以解決
  - [MapVote] 多人選擇不同地圖會等較久，若強制進行遊戲可能造成無法選擇地圖
  - [REPOGambling] 有時會沒有套用至客戶端，或是要到卡車後才會生效
  - [UniquePotions] 多人時，有時會無法抓取已經儲存在煉金鍋裡的材料
  - [顯示相關] 在遊戲過程中更換顏色會有顯示問題，至下一關即會恢復
  - [漢化相關] 吃的升級超過 12 項時，中文顯示會錯誤，變回英文。

</details>
